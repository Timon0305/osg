Straight set: is an ordinary set where you do a set of one exercise, rest and proceed to the next set.
Warm up set: perform one or several sets with low weight and intensity prior to your work sets.
Test AMRAP set: is a test set where you perform as many reps as possible with maximum effort, used to calculate your one repetition maximum (1 RM).
Pyramid set: is a collection of sets, of the same exercise, that start with a light weight and higher reps, building up to a heavier weight and fewer reps
Super set: when doing two exercises that target opposing muscle in a row with minimal rest between them.
Giant set: is a circuit of three or more exercises for one muscle group performed one after another with little to no rest in between.
Pre-exhaust set: when doing an isolation or 'single-joint' exercise followed by one compound or 'multiple-joint' movement.
Drop set: performing multiple sets of an exercise to technical failure with successively lighter loads and little to no rest.
